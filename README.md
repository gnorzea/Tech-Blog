# Tech-Blog

This project is a CMS-style blog site similar to a WordPress site, where developers can publish their blog posts and comment on other developers's blog posts as well.

## Technologies Used 
    * Handlebars.js
    * MySQL
    * Sequelize
    * express-session 
    * bcrypt
